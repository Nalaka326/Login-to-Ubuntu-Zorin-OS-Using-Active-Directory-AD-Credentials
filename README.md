# Login-to-Ubuntu-Zorin-OS-Using-Active-Directory-AD-Credentials
This guide explains how to join Ubuntu/Zorin OS to a Windows Active Directory domain using SSSD, Realmd, and Kerberos. It provides a complete step-by-step process for installing required packages, joining the domain, enabling home directory auto-creation, verifying Kerberos authentication, and ensuring GUI login works for AD users.
